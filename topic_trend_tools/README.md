## 时间影响力计算工具
事件影响力工具用于计算每个事件的影响力，并最终存储到时间记录表中。

## 使用说明
通常设置的任务的定时执行，但是特殊情况下可以执行一次来补充数据。
### 定时执行命令
使用
`$ crontab -e`
编辑每天的什么时间定时执行，示例如下：
```
# 设置定时运行
30 3 * * * sh ${path}/start
```
### 一次执行命令
`./start`
默认计算昨天的所有事件的影响力情况。

`./start <date>`
计算指定日期<date>的所有事件的影响力情况，其中<date>表示日期，格式为:yyyy-MM-dd，如：2016-07-09
## 配置说明
详细见config.properties。
